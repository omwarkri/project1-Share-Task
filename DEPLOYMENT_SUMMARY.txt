## 🚀 QUICK START COMMANDS

Start All Containers:
$ docker compose up -d

View Container Status:
$ docker compose ps

View Logs:
$ docker compose logs -f

Stop All Containers:
$ docker compose down

Restart Services:
$ docker compose restart

Create Django Superuser:
$ docker compose exec web python manage.py createsuperuser

Access Django Shell:
$ docker compose exec web python manage.py shell

Run Migrations:
$ docker compose exec web python manage.py migrate

Collect Static Files:
$ docker compose exec web python manage.py collectstatic --noinput

Database Connection:
$ docker compose exec db psql -U postgres -d todolist_db

Backup Database:
$ docker compose exec db pg_dump -U postgres todolist_db > backup.sql

View Celery Tasks:
$ docker compose exec celery celery -A todolist events

## 🔧 TROUBLESHOOTING

Container won't start?
→ Check logs: docker compose logs web

Port already in use?
→ Change ports in docker-compose.yml

Database connection error?
→ Verify db container: docker compose logs db
→ Wait longer for db initialization

Celery not processing tasks?
→ Check redis: docker compose logs redis
→ Check celery: docker compose logs celery
