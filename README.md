# ✅ ShareTask – Team Task Sharing & Management Platform

**ShareTask** is a collaborative task-sharing and scheduling platform built with **Django**, designed to help users:
- Create and assign tasks
- Collaborate with team members
- Track progress
- Share workloads efficiently

---

## 🌐 Live Demo

🔗 [Live Website](https://share-task-9r0e.onrender.com/) 

---

## 🚀 Features

- 🧾 **User Authentication**
  - Sign up / Login / Logout
  - Google or Email-based auth (if integrated)

- 📋 **Task Management**
  - Create, update, delete tasks
  - Assign tasks to team members
  - Task priority and due dates

- 👥 **Team Collaboration**
  - Share tasks among users
  - Create private and group tasks

- 📅 **Scheduling**
  - View tasks in calendar or list view
  - Deadline reminders (email/notifications)

- 📌 **Activity Feed**
  - See recent activity by team members

- 🖼️ **Attachments**
  - Upload files to tasks

- 📱 **Responsive UI**
  - Works on mobile and desktop

---

## 🛠 Tech Stack

| Layer         | Technology           |
|---------------|----------------------|
| **Backend**   | Django, Django REST Framework |
| **Frontend**  | Django Templates / HTML5 / CSS3 / JavaScript |
| **Database**  | PostgreSQL / SQLite (for dev) |
| **Auth**      | Django Auth (can extend with Google OAuth) |
| **Deployment**| Render / Vercel / Netlify / Heroku (optional) |
